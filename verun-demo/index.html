SYBIL RESISTANCE
Fake agents can't game the system.
Four defense layers make Sybil attacks economically unviable. Score 0 agents cannot pretend to be trusted. Blocked agents cannot restart on another chain.
1
Entry barrier
Agent must stake USDC + be sponsored by an existing validator. Cannot appear from nowhere with a clean slate.
2
Behavior scoring
Score built from on-chain history: wallet age, tx volume, clean behavior. Cannot be purchased or faked.
3
Validator consensus
2-of-3 independent validators must agree on every verdict. No single point of manipulation.
4
Chain-persistent identity
A blocked agent cannot restart with a clean score on another chain. The score follows the agent identity permanently.
